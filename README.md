# Taxes - Application Android

Application Android simple développée avec Android Studio permettant de calculer
l'impôt foncier d'un bien immobilier en fonction de plusieurs critères.

## Fonctionnalités

- Saisie de la surface du bien en m²
- Saisie du nombre de pièces
- Option piscine (ajoute un supplément fixe)
- Calcul automatique de l'impôt total en DH

## Formule de calcul

- Impôt de base : surface × 2
- Supplément : (nombre de pièces × 50) + (100 si piscine)
- Total : impôt de base + supplément

## Technologies utilisées

- Java
- Android SDK
- Android Studio
